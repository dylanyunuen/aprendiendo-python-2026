---

### 🧩 El Desafío

Vas a simular la venta de un **Combo Explosivo** que incluye:

- 🍟 Papas
- 🍗 Alitas
- 🥤 Refresco

Tu programa debe:

1. Crear variables para el precio de cada producto.
2. Crear variables para la cantidad de cada producto.
3. Calcular el total de cada producto (precio × cantidad).
4. Calcular el total general sumando todos los productos.
5. Mostrar todo en forma de ticket profesional.

---

### 💻 Ejemplo de estructura

No lo copies exactamente… pero inspírate:

```python
# Reto 1 - Combo Explosivo

precio_papas = 60
precio_alitas = 120
precio_refresco = 35

cantidad_papas = 2
cantidad_alitas = 1
cantidad_refresco = 3

total_papas = precio_papas * cantidad_papas
total_alitas = precio_alitas * cantidad_alitas
total_refresco = precio_refresco * cantidad_refresco

total_general = total_papas + total_alitas + total_refresco

print("===================================")
print("        COMBO EXPLOSIVO")
print("===================================")
print("Papas: $", total_papas)
print("Alitas: $", total_alitas)
print("Refresco: $", total_refresco)
print("-----------------------------------")
print("TOTAL GENERAL: $", total_general)
print("===================================")
```
