# 🏆 Misión 1: Mi Tarjeta de Programador

¡Excelente, Dylan! 🚀  

Ahora que sabes usar **cajas mágicas (variables)**, vamos a crear algo importante:

Tu primera tarjeta oficial de programador 😎💻

---

## 🧩 El Ejercicio (Misión Real)

Vas a crear un programa que guarde información sobre ti usando variables y luego la muestre en pantalla.

Tu programa debe:

1. Crear una variable con tu nombre.
2. Crear una variable con tu edad.
3. Crear una variable con tu comida favorita.
4. Crear una variable con tu videojuego favorito.
5. Mostrar toda la información usando `print()`.

---

## 💻 Ejemplo de inspiración

No lo copies igual… ¡mejóralo!

```python
# Misión 1 - Mi Tarjeta de Programador

nombre = "Dylan"
edad = 9
comida_favorita = "Papas y Alitas"
videojuego_favorito = "Minecraft"

print("================================")
print("   TARJETA OFICIAL PROGRAMADOR  ")
print("================================")
print("Nombre:", nombre)
print("Edad:", edad, "años")
print("Comida favorita:", comida_favorita)
print("Videojuego favorito:", videojuego_favorito)
print("================================")
```
