¡Hola, Dylan! 🎉

Ahora aprenderás a darle **memoria** para que pueda guardar información como si tuviera cajas mágicas.

---

### 🎯 Objetivo de la Lección
Aprender a usar **variables** para guardar texto y números, y luego mostrarlos en pantalla usando `print()`.

---

### 📚 La Lección: ¿Qué es una Variable?

Imagina que una variable es una **caja mágica con nombre** 📦✨  

Dentro de esa caja puedes guardar:
- 📝 Texto (como tu nombre)
- 🔢 Números (como tu edad)
- 💰 Cantidades (como precios)

Cuando quieras usar lo que guardaste, solo dices el nombre de la caja.

---

## 📝 Variables de Texto

El texto siempre va entre comillas `" "`.

```python
nombre = "Dylan"
comida = "Papas y Alitas"

print(nombre)
print(comida)
```

Aquí creamos dos cajas mágicas:
- `nombre` guarda "Dylan"
- `comida` guarda "Papas y Alitas"

---

## 🔢 Variables de Números

Los números **NO llevan comillas**.

```python
edad = 9
precio = 120

print(edad)
print(precio)
```

Si pones comillas en un número, el robot pensará que es texto 😮

---

## 🧪 Mezclando Texto y Variables

También puedes combinar texto con variables:

```python
nombre = "Dylan"
edad = 9

print("Hola, soy", nombre)
print("Tengo", edad, "años")
```

El robot junta todo y lo muestra bonito en pantalla.

---

### 🧠 ¿Qué estamos practicando?

- 📦 Crear cajas mágicas con nombre.
- 🔤 Diferenciar texto (con comillas) y números (sin comillas).
- 🖥 Mostrar información usando `print()`.
