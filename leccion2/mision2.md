---

## 🧩 El Ejercicio (Misión Real)

Vas a crear un pequeño sistema que simule un ticket de compra.

Tu programa debe:

1. Crear una variable para el nombre del cliente.
2. Crear una variable para el producto.
3. Crear una variable para el precio.
4. Crear una variable para la cantidad.
5. Crear una variable llamada `total` que multiplique precio × cantidad.
6. Mostrar el ticket completo en pantalla.

---

## 💻 Ejemplo de inspiración

```python
# Misión 2 - Ticket de Venta

cliente = "Carlos"
producto = "Papas y Alitas"
precio = 120
cantidad = 2

total = precio * cantidad

print("===================================")
print("        TICKET DE VENTA")
print("===================================")
print("Cliente:", cliente)
print("Producto:", producto)
print("Precio unitario: $", precio)
print("Cantidad:", cantidad)
print("-----------------------------------")
print("TOTAL A PAGAR: $", total)
print("===================================")
```

---

## 🧠 ¿Qué estamos practicando?

- 📦 Crear varias variables.
- 🔢 Multiplicar números usando `*`.
- 🧮 Guardar el resultado en otra variable.
- 🖥 Mostrar información organizada como un sistema real.

### 🎮 Desafío Extra (Modo Hacker 😎)

Intenta cambiar:

- El precio
- La cantidad

Y verifica que el total cambie automáticamente.

Si el total cambia solo…  
¡Eso significa que entendiste cómo funcionan las variables! 🧠🔥
