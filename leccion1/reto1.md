## 🧩 El Ejercicio (Misión Real)
Tu objetivo es crear el Menú de Bienvenida para el sistema de ventas de Papas y Alitas. El programa debe mostrar un cartel profesional que los clientes verían al abrir la aplicación.

Tu cartel debe tener:
1. Una línea decorativa arriba (puedes usar ======= o *******).
2. El nombre del negocio: SISTEMA DE VENTAS: PAPAS Y ALITAS.
3. Un dibujo pequeño hecho con símbolos que represente la comida (ejemplo: unas papas fritas o una alita).
4. Un mensaje de: ¡BIENVENIDOS Y BUEN PROVECHO!.
5. Una línea decorativa abajo para cerrar.

---

## 💻 Ejemplo de inspiración

```python

```

## 🧠 ¿Qué estamos practicando?
- Orden: Que las líneas queden derechitas dentro de las comillas.
- Precisión: No olvidar ningún paréntesis ().
- Creatividad: Usar símbolos para que el negocio de papá se vea genial.
