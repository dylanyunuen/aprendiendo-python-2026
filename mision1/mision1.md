# 🚀 Misión 1: El Robot que Habla (Versión Extendida)

### 🧩 El Ejercicio
Tu misión es escribir un código que haga tres cosas:
1.  Saludar al mundo.
2.  Decir cuántos años tienes.
3.  Revelar cuál es tu superpoder favorito.

---

### 💻 El Código de la Misión
Abre tu editor (Pydroid 3 o Replit) y escribe exactamente esto:

```python
# Misión 1: El Robot que Habla
# (Las líneas que empiezan con # son notas secretas para nosotros, el robot las ignora)

print("¡Hola mundo! Soy Dylan, un nuevo programador.")
print("Tengo 9 años y pronto cumpliré 10.")
print("Mi superpoder favorito es la invisibilidad.")
```
