# 🚀 Misión 1: El Robot que Habla

¡Hola, Dylan! Bienvenido a tu primera misión secreta. Hoy vas a aprender a darle voz a tu tablet usando el lenguaje de los robots: Python.

---

### 🎯 Objetivo de la Misión
Lograr que el robot (tu tablet) muestre mensajes personalizados en la pantalla para que todo el mundo sepa que ya eres un programador.

---

### 📚 La Lección: El comando print()

Imagina que el robot es muy obediente, pero un poco distraído. Para que diga algo, tienes que usar una palabra mágica llamada `print()` (que en inglés significa "imprimir" o "mostrar").

Reglas de oro para hablar con el robot:
1.  Los Paréntesis `( )`: Son como la boca del robot. Lo que esté adentro es lo que va a decir.
2.  Las Comillas `" "`: Son súper importantes. Le dicen al robot: *"Oye, esto es un mensaje de texto, no una instrucción interna"*. Si no pones comillas, el robot se confunde y marca error.

---

### 🧩 El Ejercicio
Tu misión es escribir un código que haga tres cosas:
1.  Saludar al mundo.
2.  Decir cuántos años tienes.
3.  Revelar cuál es tu superpoder favorito.

---

### 💻 El Código de la Misión
Abre tu editor (Pydroid 3 o Replit) y escribe exactamente esto:

```python
# Misión 1: El Robot que Habla
# (Las líneas que empiezan con # son notas secretas para nosotros, el robot las ignora)

print("¡Hola mundo! Soy Dylan, un nuevo programador.")
print("Tengo 9 años y pronto cumpliré 10.")
print("Mi superpoder favorito es la invisibilidad.")
```
